# ProjetoFintech

MVP de **startup de gerenciamento financeiro**, desenvolvido em **front-end** com foco em **lógica de aplicação, estrutura de código e fluxos**, e não em refinamento de UI.

## Contexto
Este projeto representa um **MVP funcional**, onde a prioridade foi validar
estrutura, organização do código e regras de negócio.
A interface visual é propositalmente simples, servindo apenas como camada de apoio.

## Objetivo
Simular a base de uma aplicação de gerenciamento financeiro, permitindo:
- organização e manipulação de dados financeiros
- implementação de fluxos principais da aplicação
- validação de estrutura para evolução futura do produto

## Abordagem
- Código front-end orientado à lógica e fluxo
- Separação clara de responsabilidades
- Estrutura preparada para escalar UI e integrações posteriormente
- Foco em clareza e manutenção do código

## O que foi implementado
- Estrutura base do front-end
- Lógica para manipulação e organização de dados financeiros
- Fluxos principais do MVP

## Stack
- JavaScript / TypeScript
- Framework front-end (conforme projeto)
- Ambiente de desenvolvimento local
